# Resume Classifier System

Machine Learning based Resume Classification using TF-IDF and Naive Bayes.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## Description
This project classifies resumes into job categories using Machine Learning.
TF-IDF is used for text vectorization and Naive Bayes for prediction.

## How to Run

pip install pandas scikit-learn streamlit

python train_model.py  
python -m streamlit run app.py
